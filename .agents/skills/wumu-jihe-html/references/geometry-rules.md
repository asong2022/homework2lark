# 几何语义、识图与 OCR 纠错

## 1. 建模清单

先建立四张表：

- 对象：点、直线、射线、线段、圆、弧、多边形。
- 定义：中点、垂足、圆心、切点、交点。
- 约束：共线、共圆、平行、垂直、相等、比例、角度、相切。
- 依赖：自由对象 → 派生对象，保证动态重算顺序无环。

不要从示意图擅自增加“水平、竖直、等长、垂直”等条件。

## 2. 常见中文 OCR 修正

| OCR | 候选 | 判断依据 |
|---|---|---|
| `◎0`、`O0` | `⊙O` | 圆符号与字母 O |
| `4ABC`、`AABC` | `△ABC` | 三角形符号 |
| `PC2` | `PC²` | 长度平方 |
| `PA:PB` | `PA·PB` 或 `PA/PB` | 点幂常为乘积；结合公式与图 |
| `90` | `⊙O` 或 `BO` | 看前后是否为“与…相交”或线段名 |
| `L`、`1` | `⊥` | “于点 H”常对应垂足 |
| 人名样字符 | 点字母 | 与图中标签及上下文互证 |

纠错优先级：数学恒真/定义一致性 > 图中拓扑 > 文字字形相似 > 图像比例。

## 3. 歧义等级

- **必须确认**：改变对象身份、连接关系、交点所在对象、平方/乘积/比例、垂直/平行、线段或延长线范围。
- **可自行推断并说明**：明显 OCR 字符，且图片和多个约束唯一支持同一结果。
- **无需确认**：配色、留白、标签偏移、整体旋转缩放、示意比例。

确认时只列关键项：`原文 → 推断 → 依据 → 若错误会造成的影响`。

## 4. 数值谓词

按图形尺度 `S=max(1, 直径或包围盒尺度)` 使用相对容差，通常 `ε=1e-7*S`：

- 共线：二维叉积接近 0。
- 垂直：方向向量点积接近 0。
- 平行：方向向量叉积接近 0。
- 圆上：点到圆心距离与半径之差接近 0。
- 中点：点等于两端点坐标平均值。
- 等距：两距离之差接近 0。
- 点幂：两边代数值之差接近 0。

同时检查对象范围：`on_segment`、`on_ray`、延长线方向、交点是否有限，以及退化情形（重合点、零半径、平行无交点）。

## 5. 版式求解

先固定数学上可自由规范化的量：平移、旋转、等比缩放。再选择一个非退化参数获得清晰布局。数学坐标与 SVG 屏幕坐标分离；用统一变换处理 y 轴方向。标签位置属于版式层，不得反向影响几何解。

## 6. 立体图可见性

- 圆柱、圆锥等旋转体的椭圆底面必须闭合表达，不得只画一侧弧。
- 底面被实体或液体遮挡时，后半弧用虚线、前半弧用实线；无遮挡时可直接画完整椭圆。
- 侧棱端点必须与底面椭圆左右端点衔接，避免底部缺口、悬空或被矩形填充裁掉。
- 填充层先画，底面轮廓后画，确保边界不会被阴影覆盖。

### 立体语义对象

识别到标准圆柱、圆锥、正方体或长方体时使用复合对象，不拆成独立线段、椭圆和弧线：

```json
{"tag":"solid","kind":"圆柱","group":"solid-1","attrs":{"cx":300,"cy1":120,"cy2":500,"rx":90,"ry":24}}
```

- `cx`：共同中心轴横坐标。
- `cy1`：圆柱顶面中心或圆锥顶点纵坐标。
- `cy2`：底面中心纵坐标。
- `rx`：底面宽度的一半。
- `ry`：底面可见椭圆高度的一半；不确定时省略，脚本默认取 `max(4, rx×0.3)`。
- 图片中底弧断裂或缺失时按结构补全；后半弧由模板自动生成虚线，前半弧生成实线。
- 圆柱的上下椭圆同轴、同宽、扁率一致；圆锥顶点默认对准底面中心轴。整体倾斜用 `transform` 旋转，不分别扭动母线。
- 水面、尺寸线、内部图形和标签分别建模；不要把整个容器场景误识别为单个立体。

正方体、长方体使用前面中心与透视偏移，便于在编辑器中继续调长、宽、高和进深：

```json
{"tag":"solid","kind":"长方体","group":"cuboid-1","attrs":{"cx":420,"cy":340,"w":260,"h":160,"ox":90,"oy":-60}}
```

- `cx/cy`：前方面中心；`w/h`：前方面宽、高；正方体只需 `w`，`h` 自动等于 `w`。
- `ox/oy`：后方面相对前方面的透视偏移；一般取右上方向。被遮挡的后棱由模板自动绘制为虚线。
- 立体的可见性以三维拓扑为准；不要照搬截图中被遮挡、断裂或重叠的棱线。

## 7. 对象选择优先级

- 标准几何结构优先使用可参数化对象：圆、椭圆、半圆、多边形、矩形、圆柱、圆锥。
- 长度或数量标注优先使用箭头线；垂直关系使用直角标记；角度关系使用角对象。
- 同一构造的主体、端点和标签设置同一 `group`；水面、辅助线等需要独立调整的对象不要误锁进主体。
- 只有无法可靠参数化的手绘痕迹才使用 `polyline` 或画笔路径。
